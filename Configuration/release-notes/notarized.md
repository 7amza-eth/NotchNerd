## Install

NotchNerd is signed with a Developer ID and **notarized by Apple**, so it just opens.

1. Download **`NotchNerd.dmg`** below and open it.
2. Drag **`NotchNerd.app`** into your **Applications** folder.
3. Launch it. NotchNerd is a **menu-bar app — no Dock icon**; look at your menu bar / the notch.
4. Grant **Accessibility** + **Automation** when prompted (System Settings → Privacy & Security).

> Prefer not to use the DMG? The **`NotchNerd.zip`** asset contains the same notarized app — unzip and drag it to /Applications.

The Claude Code agent monitor is **off by default** — enable it in **Settings → Agent**. Once installed, future versions update automatically.
